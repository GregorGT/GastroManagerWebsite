Product

Gastronome is a POS software targeting restaurants. It runs on Windows, Linux, Mac and also on Android(Mobile app). It has the following features:
  • Create and manage digital restaurant menu (Server and Android side)
  • Order food (printed to a printer)
  • Order drinks (printed to a printer)
  • Create bills and payments (printed to a printer)
  • Create and manage restaurant layout
  • Manage bookings
  • Create reports
    •

    Infographic

    Digitial menu

    The software comes with two ordering screens. One to order from the server(terminal) and one from the android app(tablet). The menus of these screens are fully customizable. Once serving a client is done the ordered items will be printed to the kitch, bar or other printers.





















    Restaurante Layout

    The software comes with a layout manager that allows editing the restaurant layout and define where tables and floors are placed within the restaurant.

































    Bookings

    The software comes with a bookings manager that lives on the server. View and edit existing reservations or check how busy the restaurant is.

















    Payments

    The software comes with two payments screens. One for the terminal(server) and one for android. Create and print bills from here.





































    Reports

    Financial reports for any time period are possible.
